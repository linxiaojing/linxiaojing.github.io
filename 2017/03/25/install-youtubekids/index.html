给电视盒子安装Youtube kids

一直以来都对培养孩子的多种语言能力很是犹豫，什么时候让孩子学习第二种语言最好呢？现在就面临着是培养方言还是普通话为主，很多人就有不同的看法，我们家是以普通话培养孩子主。而第二种语言，即英语，什么时间接触最好呢？

上个月我听了知乎LIVE的一个《如何培养优秀的双语宝宝》小讲里，主讲人韩东燃是哈佛大学医学院博士后，他和其妻子orangesky正在国外从事幼儿教育。其提的几点意见，我还是表示认同的。如，成人是通用一对一的关系进行规则记忆，要达到母语水平是比较难的。而通过学习方式的改变，为了达到母语水平，在孩子的时候就开始学习就很有优势。因为孩子早期更多的依赖记忆，他把语言和语境一起记忆下来，能过该语境出现什么语言来模仿和学习。那如果家长第二语言不好情况下，如何让孩子学第二语言，其方法中就提到了比如看动画片，接触一些有趣的语境。

在国内要购买比较好的英语的动画片，比较难。一是想高清、二是想内容要合适。而国外就有非常多不错的英语动画片，像最近比较流行的Peppe Pig（国内叫小猪佩奇），还有maisy mouse、penelope、thomas the train、team umizoomi、paw patrol等。而这些在Youtube kids的这个儿童APP里面就有，还是免费且高清的。所以我想了些办法给自己先前的乐视盒子进行了安装。

首先盒子要进行固件更新，更新可以ROOT的版本固件，找了几个版本，后来使用的是乐视盒子041S版本，然后安装相应的google框架、google服务、google player，但是发现找了好几个google框架有的都不能用，最后找到个可以使用的版本，非常折腾。后来发现国内的CIBN微视听的APP也有小猪佩奇的节目，高清的只有中文版的，英文版的资源少而且不清晰。其实为了免于折腾，可以直接某宝上购买海美迪Q5的电视盒子（这款不错，某宝上售价是699元），可以直接帮你升级最新固件，支持IOS 10 airplay播放，而且提供Google play、youtube、youtube kids的安装。

当然了，有个问题，即从小看动画片对孩子的视力是否不好，让很多家长觉得困惑的。小讲里也有提到过，我们读书的那时候发现，常看书又没有经常在外面活动的孩子就容易近视，而那些读书差，但常在外面撒野的孩子，尽管也会常到网吧玩游戏玩很久，但他们的眼睛就没近视。简单说，要保护好视力，就是让小孩在外面活动的时间和在室内看书看电视的时间要协调好，时间的控制和安排很重要，多些户外活动，让眼睛的焦距可以不断长短变换，就会起到放松眼睛的作用。
